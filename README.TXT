Challenge Name :Hidden Key
Difficulty: Easy
Author: BLAZE
Description:
The intercepted message appears to have been encrypted using a 
classical cipher, but the real challenge isn't the ciphertext. 
The encryption key has been concealed in plain sight after undergoing 
a simple transformation. Recover the key, reverse the process, and 
use it to reveal the hidden flag.
